# AI-aides
基于GPT-3和uni-app实现的人工智能助手。

本项目是GPT-3模型的一个封装，借助uni-app和flask，实现了无需翻墙，在移动端即可畅玩，尽管GPT-3模型与ChatGPT相比，智能程度上弱了不少，但因为有官方API的加持，响应速度更快，也能满足日常的玩耍等需求。
